# Fullstack App

## Instructions to start the project

**For the server**

Create a `.env` file
```
PORT=5000
```

```sh
$ cd react-day4-server-starter	
$ npm install

# To launch the server
$ npm run dev
```

**For the client**
```sh
$ cd project-management-client
$ npm install

# To launch the client
$ npm start
```
